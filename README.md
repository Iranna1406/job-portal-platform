# job-portal-platform
A Secure, Auto-Matching Job Portal System 
This project is a lightweight, full-stack Job Portal designed for secure and efficient recruitment. It features an automated resume-parsing engine, profile-matching algorithms, and a secure SQLite-backed database system.  
Key FeaturesAutomated Profiling: 
Uses an AI-powered parsing engine to analyze resumes and extract key skills and work experience.  
Intelligent Matching: Computes match scores between candidate skill sets and job requirements to streamline HR screening.  
Secure Authentication: Includes dynamic password-visibility toggles and strict input validation to prevent transcription errors.  
Lightweight Architecture: Built using Python's BaseHTTPRequestHandler and an embedded SQLite database, requiring no complex server setup
Technology StackBackend: Python (SQLite3, HTTP server).  Frontend: HTML5, Tailwind CSS, Vanilla JavaScript.  Visualization: D3.js and Recharts modules for visual analytics.  
## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Iranna1406/job-portal-platform.git
```

### Open the Project

```bash
cd job-portal-platform
```

### Start the Server

```bash
python server.py
```

Open your browser and visit:

```
http://127.0.0.1:3000
```
Recommended browser: Firefox
